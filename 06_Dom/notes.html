// // 👇 Browser internally bana deta hai ek Tree (Nodes ke form mein): // Document // └── html // └── body // ├── h1 // └── p // ============================================================ // 🌳 MAIN COMPONENTS OF DOM // ============================================================ // 1️⃣ Document → Root object (window.document se milta hai) // 2️⃣ Elements → Tags (div, h1, p, span, etc.) // 3️⃣ Attributes → class, id, src, href, etc. // 4️⃣ Text Nodes → Element ke andar ka actual text part // 5️⃣ Comments → // ============================================================ // 🧭 ELEMENT SELECTION (DOM ACCESS METHODS) // ============================================================ // 🎯 1. getElementById("idName") // -> Sirf ek element deta hai (unique ID). // const title = document.getElementById("title"); // 🎯 2. getElementsByClassName("className") // -> Similar elements (class based) deta hai (HTMLCollection form mein). // const boxes = document.getElementsByClassName("card"); // 🎯 3. getElementsByTagName("tagName") // -> Tag ke naam se select karta hai (HTMLCollection). // const paras = document.getElementsByTagName("p"); // ⚡ HTMLCollection = Live Collection (real-time update hoti hai) // 🎯 4. querySelector("CSS-selector") // -> Pehla matching element return karta hai. // const heading = document.querySelector(".heading"); // 🎯 5. querySelectorAll("CSS-selector") // -> Sab matching elements return karta hai (NodeList form mein). // const allBtns = document.querySelectorAll(".btn"); // ⚡ NodeList = Static hoti hai (live update nahi hoti) // ✅ PRO TIP: // querySelector() aur querySelectorAll() zyada flexible aur modern hain 🔥 // ============================================================ // 🧩 MODIFYING ELEMENTS (CONTENT & ATTRIBUTES) // ============================================================ // 📝 textContent → Pure text change karta hai (hidden text bhi include karta hai) // title.textContent = "Welcome to DOM World 🌍"; // 📝 innerText → Sirf visible text change karta hai (CSS hidden ignore karta hai) // title.innerText = "Learning DOM Concepts ✨"; // 📝 innerHTML → HTML structure ke andar change karne ke liye // title.innerHTML = "DOM is Powerful 💪"; // ============================================================ // 🏷️ ATTRIBUTE METHODS // ============================================================ // element.getAttribute("attributeName") // element.setAttribute("attributeName", "value") // Example: // const link = document.querySelector("a"); // link.getAttribute("href"); // get value // link.setAttribute("href", "https://google.com"); // change value // ============================================================ // 🎨 STYLING ELEMENTS (INLINE STYLES) // ============================================================ // const heading = document.querySelector("h1"); // heading.style.color = "tomato"; // heading.style.backgroundColor = "#222"; // heading.style.padding = "12px"; // heading.style.borderRadius = "10px"; // ✅ PRO TIP: Inline styles JS se apply karte waqt CSS property ko camelCase mein likho. // ❌ background-color → ✅ backgroundColor // ============================================================ // 🏗️ CREATING & ADDING NEW ELEMENTS // ============================================================ // const newDiv = document.createElement("div"); // Step 1: Create // newDiv.className = "card"; // Step 2: Add class // newDiv.id = "uniqueDiv"; // Step 3: Add ID // newDiv.setAttribute("title", "Created via JS");// Step 4: Set attribute // newDiv.innerText = "Hello! I'm a new Div 🌈"; // Step 5: Add text // document.body.appendChild(newDiv); // Step 6: Add to body (at end) // ✅ appendChild() → Adds element at end // ✅ prepend() → Adds at beginning // ✅ before() / after() → Add relative to existing element // ============================================================ // 🗑️ REMOVING ELEMENTS // ============================================================ // const element = document.querySelector(".removeMe"); // element.remove(); // Directly removes the element // Or (older way): // element.parentNode.removeChild(element); // ============================================================ // 🔁 TRAVERSING THE DOM // ============================================================ // const parent = document.querySelector(".parent"); // 👶 parent.children → HTMLCollection of all child elements // 👆 parent.firstElementChild → First child // 👇 parent.lastElementChild → Last child // 🔝 child.parentElement → Parent node // 👉 element.nextElementSibling → Next element // 👈 element.previousElementSibling → Previous element // ============================================================ // ⚡ CLASS MANIPULATION // ============================================================ // element.classList.add("active") // element.classList.remove("hidden") // element.classList.toggle("dark-mode") // toggle = add/remove on click // ============================================================ // 💥 DOM EVENTS (INTERACTIVITY) // ============================================================ // 🎯 Method 1: Inline (Not Recommended) // // 🎯 Method 2: JS Event Listener (Recommended) // const btn = document.querySelector("#btn"); // btn.addEventListener("click", () => { // console.log("Button Clicked 🎯"); // }); // ✅ Common Events: // - click // - dblclick // - mouseover / mouseout // - keydown / keyup // - input / change / submit // - focus / blur // ============================================================ // 🧨 EVENT OBJECT (e) // ============================================================ // btn.addEventListener("click", (e) => { // console.log(e.target); // jis element pe click hua // console.log(e.type); // event type (like click) // }); // ============================================================ // 🔄 EVENT BUBBLING & CAPTURING (ADVANCED) // ============================================================ // Bubbling = Event niche se upar jaata hai (child → parent) // Capturing = Event upar se niche aata hai (parent → child) // Example: // document.querySelector(".child").addEventListener("click", () => { // console.log("Child Clicked"); // }); // document.querySelector(".parent").addEventListener("click", () => { // console.log("Parent Clicked"); // }); // Output order (by default) → Child → Parent (Bubbling) // Bubbling rokne ke liye: // e.stopPropagation(); // ============================================================ // 🧠 EVENT DELEGATION (SUPER USEFUL) // ============================================================ // Instead of adding listener to each child, add it to parent: // document.querySelector(".list").addEventListener("click", (e) => { // if (e.target.tagName === "LI") { // console.log("Clicked:", e.target.textContent); // } // }); // ✅ Use delegation jab dynamically elements add kar rahe ho. // ============================================================ // ⚙️ DOCUMENT METHODS // ============================================================ // document.createElement() → naya element banana // document.querySelector() → element select karna // document.body.appendChild() → element add karna // document.title → page title access/change // document.URL → current page URL // document.domain → domain name // ============================================================ // 💎 BEST PRACTICES // ============================================================ // ✅ Always cache your selectors (use const) // ✅ Avoid inline JS in HTML // ✅ Use querySelector / querySelectorAll for flexibility // ✅ Use classList for toggling classes dynamically // ✅ Use event delegation for better performance // ✅ Minimize DOM reflows by batching changes // ============================================================ // 🧭 MINI REVISION SUMMARY // ============================================================ // 🌿 DOM = HTML ka JS object model // 🌿 Tree structure (nodes) hoti hai // 🌿 Hum JS se HTML elements ko access, style, modify, delete kar sakte hain // 🌿 Events se page interactive banta hai // 🌿 Advanced: Bubbling, Capturing, Delegation concepts real projects mein kaam aate hain // 🌿 Master this → Frontend ke 70% DOM + JS projects easily bana lega 🚀 // ============================================================ // 🏁 END OF DOM NOTES (Pro Edition by ChatGPT x Parth 😎) // ============================================================
